1차시 · 라이브 시연

30초 응대 스크립트

너는 병원 CS 전담 직원이야.
진료 대기 2시간으로 화가 난 환자가 접수창구에서 항의하고 있어.
공감하고 대안을 제시하는 응대 스크립트를 3가지 작성해줘.
각 3문장 이내, 경어체 사용.
2차시 · 민원 응대

민원 유형별 스크립트

너는 [기관명] 고객서비스 CS 전담 직원이야.
민원 유형: [대기/수납/의료진 태도/시설/의료행위]
상황: [예. 진료 대기 2시간으로 화가 난 환자]
공감 → 수용 → 해결 → 마무리 4단계 구조로 응대 스크립트 3가지를 작성해줘.
각 3문장 이내, 경어체 사용.
2차시 · 강성민원

공감 + 인정 + 해결

역할: 병원 CS 전담 응대 직원
상황: 환자 보호자가 수술 후 설명 부족을 이유로 격한 항의 중
요청: 공감 + 상황 인정 + 해결 제시 구조로 응대 스크립트 2가지를 작성해줘.
조건: 각 4문장 이내, 경어체, 단호하지만 공감을 담아줘.

너는 [기관명]의 고객서비스(CS) 전담 AI 어시스턴트야.
